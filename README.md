# react-native-leveldb

Superfast React Native bindings for LevelDB

## Installation

```sh
npm install react-native-leveldb
```

## Usage

```js
import Leveldb from "react-native-leveldb";

// ...

const result = await Leveldb.multiply(3, 7);
```
